Konfigurasi Diperlukan

Isi SUPABASE_URL dan SUPABASE_ANON_KEY dari project Supabase kamu di bagian bawah tag <script>.

  1. Buka supabase.com → Sign Up → buat project baru
  2. Di dashboard: Project Settings → API
  3. Salin Project URL dan anon public key
  4. Buka file index.html dengan text editor, cari baris const SUPABASE_URL dan const SUPABASE_ANON_KEY, paste nilai tersebut
  5. Refresh halaman ini
